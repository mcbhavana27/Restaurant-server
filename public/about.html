The tech stack

  • The app is created using Express
  • Bootstrap is used for styling pages
  • Icons are courtesy Font Awesome
  • EJS is used for templating.
  • Mongoose is used to connect to a MongoDB server and validating data against a schema when querying
  • bcrypt is used for hashing passwords , jswebtoken is used to generate JWTs (for authentication and authorization ), multer is used to manage file uploads , morgan and winston are used for logging . You can search for these projects on Github or npmjs.com
  • The frontend which is supplied is built using React
